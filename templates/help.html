Обзор системы

Network Scanner Pro - это профессиональный инструмент для сканирования сетей, который позволяет обнаруживать активные хосты, открытые порты и получать детальную информацию о сетевых сервисах.

Основные возможности:
  • Сканирование сетей и подсетей
  • Обнаружение открытых портов
  • Получение баннеров сервисов
  • Создание скриншотов веб-страниц
  • Генерация подробных отчетов
  • Мониторинг системных ресурсов

Мониторинг системы

CPU

Отображает текущую загрузку процессора в реальном времени.

RAM

Показывает использование оперативной памяти.

Диск

Отображает заполненность дискового пространства.

Сеть

Показывает текущую скорость сетевого трафика в MB/s.

Создание задач сканирования

Как создать задачу:
  1. Введите сеть в формате CIDR (например: 172.30.1.0/24)
  2. Нажмите кнопку "Создать задачу"
  3. Задача будет добавлена в очередь и начнет выполняться
Поддерживаемые форматы сетей:
  • 192.168.1.0/24 - подсеть класса C (256 адресов)
  • 10.0.0.0/16 - подсеть класса B (65536 адресов)
  • 172.30.1.0/28 - малая подсеть (16 адресов)
  • 192.168.1.0/30 - минимальная подсеть (4 адреса)
Рекомендации:
  • Начинайте с малых подсетей для тестирования
  • Используйте /24 для небольших сетей
  • Избегайте сканирования больших сетей без необходимости
  • Учитывайте время выполнения при выборе размера сети

Результаты сканирования

Что сканируется:
  • Общие порты: 21, 22, 23, 25, 53, 80, 110, 143, 443, 993, 995, 8080, 8443
  • Обнаружение хостов: ICMP ping и TCP SYN сканирование
  • Баннеры сервисов: Получение информации о версиях
  • Операционные системы: Определение ОС по ответам
  • Веб-сервисы: Автоматическое создание скриншотов
Типы отчетов:
  • JSON отчет: Структурированные данные для анализа
  • Текстовый отчет: Человекочитаемый формат
  • Скриншоты: Изображения веб-страниц
  • Архив: Все результаты в ZIP файле

Управление задачами

Действия с задачами:
  • Скачать отчет: Получить результаты в ZIP архиве
  • Удалить задачу: Очистить историю и файлы
  • Просмотр прогресса: Отслеживание выполнения в реальном времени
Автоматические действия:
  • Автоматическое создание скриншотов для веб-сервисов
  • Генерация отчетов по завершении
  • Сжатие результатов в архив
  • Очистка временных файлов

Безопасность и рекомендации

Важно!
  • Сканируйте только те сети, на которые у вас есть разрешение
  • Некоторые системы могут блокировать сканирование
  • Избегайте сканирования производственных сетей в рабочее время
  • Используйте результаты только в законных целях

Поддержка

  • Проверьте логи в директории logs/

  • Перезапустите веб-интерфейс при необходимости
